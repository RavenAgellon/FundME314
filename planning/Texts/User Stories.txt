FundRaising Project User Stories

#0 As a user admin, I want to create user roles, so that there can be new roles for new purposes if needed.
#1 As a user admin, I want to view user roles, so that I can see the details of what a user role has permission to perform.
#2 As a user admin, I want to update a user role, so that it will have the latest information on the user role.
#3 As a user admin, I want to suspend a user profile, so that I can deny access to any user accounts of the user role
#4 As a user admin, I want to search for a user role, so that I can find a specific user role to manage.
#5 As a user admin, I want to create a user account, so that more users can join in
#6 As a user admin, I want to view user accounts so that I can see the details of user accounts
#7. As a user admin, I want to update a user account, so that it has the latest information on the user account.
#8 As a user admin, I want to suspend a user account, so that I can deny access to the user account.
#9 As a user admin, I want to search for user accounts, so that I can find a specific account to manage.
#10 As a user admin, I want to log into the user account, so that I can access the system
#11 As a user admin, I want to log out of the user account, so that I can leave the system

#12 As a fundraiser (also known as an FR), I want to create fundraiser activities (also known as FRAs) so that there can be new FRAs
#13 As a FR, I want to view an FRA so that I can see the details on the FRA
#14 As a FR, I want to update an FRA so that I can ensure it will have the latest information on the FRA.
#15 As a FR, I want to suspend/stop the FRA so that I can prevent any access to the FRA when it is completed
#16 As a FR, I want to search for a FRA so that I can find a specific FRA to manage.
#17 As a FR, I want to log into the user account so that I can access the system
#18 As a FR, I want to log out of the user account so that I can access the system

#19 As a donee, I want to search for FRAs so that I can find a specific FRA that I am interested in
#20 As a donee, I want to view details on FRAs so that with the information, I can decide whether to support it
#21 As a donee, I want to save FRAs to a favourites list so that I can keep track of them easily
#22 As a donee, I want to view FRAs saved in my favourites list so that I can see the list of FRAs that have been saved inside
#23 As a donee, I want to search for FRAs saved in my favourites list so that I can find a specific FRA in the list.
#24 As a donee, I want to log into the user account so that I can access the system
#25 As a donee, I want to log out of the user account so that I can leave the system

#26 As a FR, I want to view the number of times an FRA has been viewed by a done so that I can track the interest on the FRA.
#27 As a FR, I want to view the number of times an FRA has been saved so that I can monitor how many people has added the FRA in their favourites list.
#28 As a FR, I want to search for completed FRAs so that I can view the list of FRAs that are completed.
#29 As a FR, I want to view completed FRAs so that I can see the details of the completed FRA

#30 As a donee I want to search for completed FRAs so that I can view the list of FRAs that are completed
#31 As a donee, I want to view completed FRAs so that I can see the details of the completed FRA

#32 As a platform management, I want to create FRA categories so that FRAs can be viewed by its categories
#33 As a platform management, I want to view FRA categories so that I can see the list of FRAs under the FRA category
#34 As a platform management, I want to update the FRA categories so that it has the latest list of FRAs under the category
#35 As a platform management, I want to delete FRA categories so that I can remove categories that are no longer needed
#36 As a platform management, I want to search for FRA categories so that I can find a specific FRA category
#37 As a platform management, I want to generate a daily report so that I can see how the platform has performed on a daily basis
#38 As a platform management, I want to generate a weekly report so that I can see how the platform has performed on a weekly basis
#39 As a platform management, I want to generate a monthly report so that I can see how the platform has performed on a monthly basis
#40 As a platform management, I want to log into the user account so that I can access the system
#41 As a platform management, I want to log out of the user account so that I can leave the system
